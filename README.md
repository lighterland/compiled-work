# Compiled Work & Projects

A personal repository dedicated to collecting compiled academic assignments, course tasks, and independent/self-directed projects across various subjects and domains.

---

## 📁 Repository Structure

Each subject or domain is organized into its own top-level directory:

### 1. 📊 [macroeconomic](./macroeconomic)
Contains assignments and interactive materials related to Macroeconomics.
- **Main Entry Point**: [macroeconomic/index.html](./macroeconomic/index.html) – The main web interface for the macroeconomics tasks.
- **Template**: [macroeconomic/template.html](./macroeconomic/template.html)

### 2. 🌍 [economic-geography](./economic-geography)
Contains work related to Economic Geography.
- **Interactive Scatter Plot**: [task_5_1_lorenz_scatter.html](./economic-geography/task_5_1_lorenz_scatter.html)
- **Centers of World Economy**: [task_5_7_centers_of_world_economy.html](./economic-geography/task_5_7_centers_of_world_economy.html)
- **Visual Maps**: Includes world contour maps and other geography aids.

### 3. 💻 [digital-skills](./digital-skills)
Contains coursework and laboratory assignments for Digital Skills.
- **Organization**: Divided by semester (e.g., [2nd-semester](./digital-skills/2nd-semester)).
- **Contents**: PowerPoint presentation slides (`.pptx`) and Excel spreadsheet exercises (`.xlsx`) covering various digital lab tasks.

---

## 🚀 Adding New Subjects / Projects

To expand this repository with new subjects (e.g., `physics/`, `math/`, `cs/`, `personal-projects/`):

1. **Create a Top-Level Folder**: Use lowercase and kebab-case for naming (e.g., `computer-science/`).
2. **Add Your Files**: Place all subject-specific assignments, code, or documentation inside that folder.
3. **Update the README**: Document the new subject and its entry points in the **Repository Structure** section above.
