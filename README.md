# Compiled Work Repository

This repository collects compiled assignments and related materials from multiple subjects. Each subject should have its own top‑level folder. For the current subject **macroeconomic**, the main entry point is:

- `macroeconomic/index.html` – the HTML file for the macroeconomics assignment.

Add additional subject folders (e.g., `physics/`, `math/`, `cs/`) as needed and place their respective files inside.
